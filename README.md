# Hacker news webs scraper

## Requirements

- node 20

## Instructions

### Backend

- Nav to backend dir
- Add a .env file just like the example file .env.template
- install dependencies
  ``` npm i ```
- lift the server
  ```npm run dev```

### Frontend

- Nav to frontend dir
- install dependencies
  ``` npm i ```
- lift the server
  ```npm run start```
- go to http://localhost:4200 in the web browser

#### Note

To run tests use the command ```npm run test```
